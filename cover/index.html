The dataset comprises hourly aggregated profiles of prosumers throughout a one-year timeframe (2022-2023). Prosumers are individuals or entities who both consume and produce energy. This data encompasses approximately a dozen prosumers located in Vilnius, Klaipėda, and Panevėžys.

To copy all cells at once from an IPython Notebook (.ipynb) file in Visual Studio Code, you can follow these steps: Open the IPython Notebook file in Visual Studio Code. Click on the first cell in the notebook to select it. Scroll down to the last cell in the notebook. Hold down the Shift key on your keyboard. Click on the last cell in the notebook. This should select all cells between the first and last cell. Once all cells are selected, you can right-click and choose "Copy" from the context menu, or you can use the keyboard shortcut Ctrl + C (Cmd + C on macOS) to copy the selected content. Now, you can paste the copied content wherever you like using Ctrl + V (Cmd + V on macOS) or by right-clicking and selecting "Paste".
